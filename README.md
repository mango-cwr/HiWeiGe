# 套餐详情分析

在浏览器中上传套餐文件（Excel：`.xlsx` / `.xls`）并进行分析。

## 功能

- **上传套餐文件**：上传 Excel 格式的套餐/账单文件进行分析。
- **多工作表**：支持多 sheet，可切换查看。
- **套餐与接入号**：按套餐归类，展示接入号、费用明细、小计/合计；分析结果默认折叠，可点击展开。
- **搜索**：按产品名称、接入号、费用项目等关键词过滤。
- **复制接入号**：每条接入号行支持一键复制。

## 使用步骤

1. 安装依赖（在项目目录下执行）：

   ```bash
   pip install -r requirements.txt
   ```

2. 启动服务：

   ```bash
   python app.py
   ```

3. 在浏览器中打开：

   ```
   http://127.0.0.1:5000
   ```

4. 在左侧上传套餐文件，右侧会显示套餐详情分析结果（默认折叠；可点击套餐标题或「一键展开」查看明细）。

## 依赖说明

- **Flask**：Web 服务
- **openpyxl**：读写 `.xlsx`
- **pandas**：解析 Excel 并做统计

仅支持 `.xlsx`；`.xls` 需额外安装 `xlrd` 方可读取。
